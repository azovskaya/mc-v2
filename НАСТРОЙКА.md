# Настройка заведения (один раз)

Нужны: Google-аккаунт, планшет/ПК с камерой, ~15 минут.

---

## 1. Google Таблица + скрипт

1. Откройте [Google Таблицы](https://sheets.google.com) → **Пустой файл**  
2. Назовите, например: **«Столовая — питание v2»**  
3. **Расширения → Apps Script**  
4. Удалите код по умолчанию, вставьте содержимое файла `apps-script/Code.gs`  
5. Сохраните проект  
6. В списке функций выберите **`setupSheets`** → ▶ **Выполнить** → разрешите доступ  
7. Появятся листы: `Employees`, `Meals`, `Logs`, `Settings`

### Публикация

1. **Развернуть → Новое развертывание**  
2. Тип: **Веб-приложение**  
3. Запуск от имени: **Я**  
4. У кого есть доступ: **Все**  
5. **Развернуть** → скопируйте URL, заканчивающийся на `/exec`

> В настройках проекта Apps Script поставьте часовой пояс **Asia/Almaty** (или ваш).

---

## 2. Мастер в админке

1. Положите папку проекта на хостинг или запустите локально:

```bash
python3 -m http.server 8080
```

2. Откройте **admin.html**  
3. Введите:
   - URL `.../exec`
   - Название точки
   - PIN администратора (4–8 цифр)
   - По желанию — цены завтрака/обеда/ужина  
4. **Сохранить и войти**

После этого настройки лежат на сервере. Очистка браузера киоска их не сбросит (нужен только сохранённый URL — он в `localStorage` устройства; можно передать `?api=URL` в ссылке киоска).

---

## 3. Сотрудники

В админке → **Сотрудники**:

1. Включите **Камера**  
2. Введите ФИО  
3. **Сохранить с Face ID**  

Без Face ID киоск человека не узнает.

---

## 4. Киоск

Откройте **index.html** на устройстве у раздачи (лучше Chrome, HTTPS или localhost).

Сценарий:

1. Узнал лицо → **Приятного аппетита**  
2. Нажал **Спасибо за завтрак / обед / ужин** → запись в таблицу  
3. Если блюда нет → **Другое блюдо** → текст + цена → **Спасибо**

---

## Частые вопросы

| Проблема | Что сделать |
|----------|-------------|
| Модели не грузятся | Не открывайте через `file://`, только через сервер |
| Камера запрещена | Разрешите в браузере, нужен HTTPS/localhost |
| «Нет URL сервера» | Пройдите мастер в admin.html или добавьте `?api=...` |
| Забыли PIN | В Apps Script на листе Settings удалите строку `pinHash` и пройдите мастер снова |
| Фото не видны | Проверьте папку Drive `MealKioskPhotos` и доступ «по ссылке» |

## Обновление скрипта

Если меняли `Code.gs`: **Развернуть → Управление развертываниями → Изменить → Новая версия**.  
URL обычно тот же.
