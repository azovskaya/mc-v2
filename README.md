# Meal Kiosk v2 — Питание персонала

Вторая версия киоска учёта питания с Face ID.

- **Киоск** (`index.html`) — только камера и фиксация питания
- **Админка** (`admin.html`) — сотрудники, журнал, отчёты для бухгалтерии (Excel), настройки под PIN
- **Бэкенд** — Google Sheets + Apps Script

## Как пользуется официант

1. Сотрудник подходит к камере  
2. На экране: **«Приятного аппетита»** и имя  
3. Нажимает **«Спасибо за завтрак / обед / ужин»** — запись сохраняется  
4. Если основного блюда нет → **«Другое блюдо»** → что приготовили + цена → **«Спасибо»**

Никаких «Выдать», настроек и отчётов на экране раздачи.

## Быстрый старт

1. Создайте Google Таблицу → Apps Script → вставьте `apps-script/Code.gs`  
2. Выполните `setupSheets`  
3. Разверните как **веб-приложение** (доступ: Все), скопируйте URL `.../exec`  
4. Впишите URL в `js/config.js` → `apiUrl` (один раз — больше не спрашивает)  
5. Откройте `admin.html` → задайте PIN  
6. Добавьте сотрудников с Face ID  
7. Откройте `index.html` на планшете у раздачи  

Данные (сотрудники, питание, цены) живут в Google Таблице — её можно открывать как Excel.

Подробно: [`НАСТРОЙКА.md`](./НАСТРОЙКА.md)

## Локальный запуск

Нужен HTTP(S), не `file://`:

```bash
cd mc-v2
python3 -m http.server 8080
```

- Киоск: http://localhost:8080/  
- Админка: http://localhost:8080/admin.html

## Отличия от v1

| v1 | v2 |
|----|----|
| Киоск + админ в одном экране | Раздельно |
| PIN только в браузере | PIN на сервере (hash) |
| URL зашит в HTML | Один раз в `js/config.js` — на всех устройствах |
| Cooldown 60 мин | В любое время + защита от двойного тапа |
| «Доп. питание» | «Другое блюдо» с описанием и ценой |
| Кнопка выдачи по расписанию | «Спасибо за …» + тип всегда явно |

## Структура

```
index.html          киоск
admin.html          админка
js/config.js        URL сервера (заполнить один раз)
apps-script/Code.gs бэкенд → Google Таблица (Excel)
models/             Face ID модели
js/ css/            фронт
```
